# Sales Analysis Dashboard - Google Sheets 📊

**Built by:** Nikhil  
**Date:** 22nd June 2026  
**Tools:** Google Sheets, Pivot Tables, Charts, Slicers

## 🎯 Project Overview
Interactive sales dashboard analyzing ₹50.19 Lakh revenue data across 4 regions. Built from scratch in 77 minutes as Day 1 of my Data Analyst journey.

## ✨ Key Features
- **Dynamic Pivot Tables** for data aggregation
- **Interactive Charts** - Bar chart for regions, Column chart for products  
- **Slicer Integration** - Filter entire dashboard by region in 1 click
- **Professional Design** - Title, insights, and clean formatting

## 📈 Key Insight
North Region leads with ₹13.69 Lakh revenue, representing 27% of total sales.

- Business Insights Communication

## 🚀 How to Use
1. Download `sales-analysis-dashboard-nikhil.xlsx`
2. Open in Google Sheets or Excel
